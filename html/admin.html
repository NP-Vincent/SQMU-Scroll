Administrator Instructions

      
    
        
  1. Click Connect Wallet and approve access in MetaMask. The interface automatically switches to the Scroll network.
    2. 
        
  2. Use the Mint Tokens section to create new SQMU tokens. Only the contract owner can mint.
    3. 
        
  3. Use Transfer Tokens to move existing tokens to another address.
    4. 
        
  4. The Check Balance section shows balances for any address. Leave the address empty to use your connected wallet.
    5. 
        
  5. Click Disconnect when finished.
    6.
